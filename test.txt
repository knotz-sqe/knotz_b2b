1.  SRS 복사 금지 (절대 금지!!!!!)

2. TC 작성 (절차)
- Tab 이동 > 으로 표시
- 절차 3개 이상 이동 금지
- Value 설정 TC > : 사용
ex)
1. Webviewer 접속
2. Setup > Video & Audio > Camera setup
3. White balance > Mode : ATW 설정
4. Apply
...

3. TC 작성 (Result)
- 문장으로 마무리 되도록 작성
ex) ~~~영상 화질 변경 (x) > ~~~영상 화질이 변경되어야 한다. (o)


4. TC 작성 (Precondition)
- 설정되어야하는 순서로 작성 (상관있는 경우)
지금 대부분 TC 는 짧아서 괜찮지만, 길어질 경우 같은 설정을 여러번 해야하는 일이 발생하여 시간낭비를 할수있어 통일
ex)
"VBR, Record profile 체크 해제상태의 H.264 또는 H.265 Profile 생성" (x)

> 수정
1. Profile 추가
2. Codec : H.264 or H.265
3. Record Profile : Disable
4. Bitrate control : VBR

5. Flow, 중요도
- 성능으로 TC N/A 처리시 Flow "NF" 로 수정
- 중요도 무조건 "중" 으로 입력하지 말고 TC 내용에 맞는 중요도로 입력
- 성능 중요도 "하"로 통일 (성능에 따라 다른 중요도 선택해도됨)


* 지금까지는 TC검토시 검토자가 조금씩 수정했지만,
앞으로는 위에 내용대로 작성되어있지 않은 경우 검토자 수정없이 담당자에게 수정 요청예정입니다.
